# EquipmentCheckout

 GB manufacturing is having issues with lost, stolen, or damage tool

 路 The business would like to automate the checkout systems for the tools used by the employees

 路 The material warehouse is responsible for storing supplies that will be needed to complete the job

 路 There are two warehouses that store all the tools needed to perform the job. This becomes a problem because employees tend to only want to check the availability of the tools in the smallest facility which is the closest one to them

 路 Building the software will help the business problem by

 o Reducing the amount of time, it take to process an equipment checkout

 o Keeping track of inventory

  Ensuring that each employee is responsible to return the tools that are checked out

  Preventing the loss of equipment

 reduce cost of stolen equipment
